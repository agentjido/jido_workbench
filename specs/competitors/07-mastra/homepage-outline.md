# Homepage Element Outline

Reviewed: 2026-02-25
Homepage: https://mastra.ai/
Site type: Product-marketing homepage

## Top-to-bottom element map

1. Global navigation header
- Product, resources, pricing, docs/blog, auth.
- Includes visible social proof metric (GitHub stars).

2. Hero section
- H1: "Build agents with a modern TypeScript stack".
- Fast-start install command surfaced directly in hero.
- Immediate capability shortcuts (Agents, Workflows, RAG, Memory, Tools, MCP, Templates).

3. Build/iterate platform section
- Explicit "Build and iterate" framing.
- Dev server and developer studio emphasis.

4. Deploy/scale section
- "Deploy and scale" transition from local development to production.

5. Template showcase grid (high-impact visual section)
- Card grid with concrete examples:
- Browser agent, Google Sheet analysis, Coding agent, Deep research, PDF-to-audio, Text-to-SQL.

6. Ecosystem/community learning links
- Agent Book, tutorials, workshops, community programs.

7. Enterprise/contact CTA
- Contact sales / conversion endpoint near bottom.

8. Footer navigation
- Product/resources/company/utilities.

## Notes for Jido direction
- This is a primary style benchmark for Jido.
- Strongest patterns to copy: immediate install CTA, concrete template cards, and toolchain-first narrative.
